여기서 지켜야 할 사항!!

1. 패키지 명은 항상 소문자를 사용합니다.
2. 클래스 파일명은 반드시 대문자로 작성합니다.
3. 클래스 명이 파일 명이 되어야 합니다.



기본 자료형

1. 참 거짓 - Boolean
2. 문자 - Char
3. 정수 - Byte, Short, Int, Long
4. 실수 - Float, Double
